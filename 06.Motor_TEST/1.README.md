# STM32 Smart Car Control (UART Interrupt)

UART 인터럽트 수신을 이용해 스마트카를 전/후/좌/우로 제어하고,  
일정 시간 입력이 없으면 자동으로 정지

---

## 📂 프로젝트 구조
- `main.c` – 메인 애플리케이션 코드
- `main.h` – 핀 정의 (모터 드라이버, LED 등)
- `stm32xxxx_hal_*.c` – HAL 라이브러리 (STM32CubeMX 자동 생성)

---

## 🔧 주요 기능
- **UART 인터럽트 수신** – 비동기 입력 처리 (`HAL_UART_Receive_IT`)  
- **모터 제어 함수** – 전/후/좌/우/정지 각각 별도 함수로 구현  
- **자동 정지 기능** – 마지막 입력 이후 100ms 이상 경과 시 정지  
- **명령어 입력**
  | 입력키 | 동작         |
  |-------|-------------|
  | `w`   | 전진 (Forward) |
  | `s`   | 후진 (Backward) |
  | `a`   | 좌회전 (Left) |
  | `d`   | 우회전 (Right) |
  | `q`   | 정지 (Stop) |

---

## 🖼 주요 코드

### UART 인터럽트 수신 & 명령 처리
```c
void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart)
{
    last_rx_tick = HAL_GetTick();

    if(ch == 'w') { printf("forward\n"); smartcar_forward(); }
    else if(ch == 's') { printf("backward\n"); smartcar_backward(); }
    else if(ch == 'q') { printf("stop\n"); smartcar_stop(); }
    else if(ch == 'd') { printf("right\n"); smartcar_right(); }
    else if(ch == 'a') { printf("left\n"); smartcar_left(); }
    else { printf("key error\n"); smartcar_stop(); }

    HAL_UART_Receive_IT(&huart2, &ch, 1);
}
```

### 메인 루프 (자동 정지)
```c
while (1)
{
    if (HAL_GetTick() - last_rx_tick > 100)
    {
        smartcar_stop();
    }
}
```

## ▶ 빌드 & 실행 방법
1. STM32CubeIDE에서 프로젝트 열기
2. Build 버튼으로 컴파일
3. 보드에 펌웨어 업로드
4. 시리얼 터미널에서 `w`, `s`, `a`, `d`, `q` 입력으로 스마트카 제어
5. 입력이 없으면 100ms 후 자동 정지 확인

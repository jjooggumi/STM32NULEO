# STM32 Dual HC-SR04 Ultrasonic Ranging

왼쪽/오른쪽 두 개의 HC-SR04 초음파 센서를 사용하여  
1초마다 양쪽 거리(mm)를 측정하고 UART를 통해 출력

---

## 📂 프로젝트 구조
- `main.c` – 메인 애플리케이션 코드
- `main.h` – 핀 정의 (TRIG, ECHO, LD2 등)
- `stm32xxxx_hal_*.c` – HAL 라이브러리 (STM32CubeMX 자동 생성)

---

## 🔧 주요 기능
- **시스템 클록 설정** – 내부 HSI 클럭 + PLL로 시스템 클록 생성  
- **TIM1 초기화** – μs 단위 정밀 시간 측정용 (Echo 펄스 폭 측정)  
- **TRIG/ECHO 제어**  
  - `trig_l()`, `trig_r()` → 초음파 송신 펄스 발생 (10μs)
  - `echo_l()`, `echo_r()` → ECHO 핀 High 시간 측정 → 거리 계산  
- **UART 출력** – 매 측정 주기마다 거리(mm) 단위로 출력  
- **오류 처리** – Echo 신호가 유효 범위를 벗어나면 `"Out of Range!"` 출력  

---

## 🖼 주요 코드

### TRIG 펄스 & ECHO 측정
```c
void trig_l(void)
{
    HAL_GPIO_WritePin(TRIG_L_GPIO_Port, TRIG_L_Pin, 1);
    delay_us(10);
    HAL_GPIO_WritePin(TRIG_L_GPIO_Port, TRIG_L_Pin, 0);
}

long unsigned int echo_l(void)
{
    long unsigned int echo = 0;
    while(HAL_GPIO_ReadPin(ECHO_L_GPIO_Port, ECHO_L_Pin) == 0){}
    __HAL_TIM_SET_COUNTER(&htim1, 0);
    while(HAL_GPIO_ReadPin(ECHO_L_GPIO_Port, ECHO_L_Pin) == 1);
    echo = __HAL_TIM_GET_COUNTER(&htim1);
    if(echo >= 240 && echo <= 23000) return echo;
    else return 0;
}
```
TRIG 핀에 10μs 펄스 출력 후 ECHO 핀 High 시간 측정 → 거리를 계산

### 메인 루프 (거리 측정 & UART 출력)
```c
while (1)
{
    HAL_Delay(1000);

    trig_l();
    echo_l_time = echo_l();
    if(echo_l_time != 0)
    {
        dist_l = (int)(17 * echo_l_time / 100);
        printf("Distance_l = %d(mm)\n", dist_l);
    }
    else
    {
        printf("Out of Range!@L\n");
    }

    trig_r();
    echo_r_time = echo_r();
    if(echo_r_time != 0)
    {
        dist_r = (int)(17 * echo_r_time / 100);
        printf("Distance_r = %d(mm)\n", dist_r);
    }
    else
    {
        printf("Out of Range!@R\n");
    }
}
```
1초 간격으로 왼쪽/오른쪽 센서 순차 측정
Echo 시간이 240~23000 μs 범위면 거리 계산, 아니면 "Out of Range" 출력

## 🔌 타이머 설정 (TIM1)
```c
htim1.Instance = TIM1;
htim1.Init.Prescaler = 64-1;  // 64MHz / 64 = 1 MHz → 1 tick = 1μs
htim1.Init.Period = 65535;
HAL_TIM_Base_Init(&htim1);
```
Echo 측정을 μs 단위로 하기 위해 1 MHz 타이머 설정

## ▶ 빌드 & 실행 방법
1. HC-SR04의 TRIG/ECHO 핀을 MCU와 연결 (왼쪽/오른쪽 각각)
2. STM32CubeIDE에서 프로젝트 빌드 & 다운로드
3. 터미널(115200bps) 열고 거리 출력 확인
3. Echo가 없으면 "Out of Range!" 메시지 확인
